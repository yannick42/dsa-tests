Splines
Goal : to test methods to draw splines, eg. (cubic) Bézier curves (1959), ...

- Using 3 "nested" linear interpolations (lerp).
- We can also use Bernstein polynomials (1912) over [0,1] to draw various types of parametric curve (quadratic, cubic, ...) including Bézier curves as a particular case.
