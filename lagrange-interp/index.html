Lagrange interpolation
An interpolating polynomial with the form of a linear combinaison:
$$L(x)=\sum\limits_{j=0}^k y_j \ell_j(x) \ \text{with coef.} \ \ell_j(x) = \prod_{\begin{smallmatrix}0\le m\le k\\ m\neq j\end{smallmatrix}} \frac{x-x_m}{x_j-x_m} \text{ and k given points } (x, y)$$
  • First discovered in 1779 by Edward Warring (1736-1798), rediscovered later by Leonhard Euler (1707-1783) as an easy consequence of a formula, published in 1783. And also published in 1795 by Joseph-Louis Lagrange (1736-1813).
  • The degree of the polynomial is 1 less than the number of points.
  • Drawbacks: if a new point is added then everything must be recalculated. If points are equally-spaced, it can produce large oscillations (Runge's phenomenon) that can be eliminated by using the Chebyshev nodes instead.
